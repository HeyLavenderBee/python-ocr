# Python OCR

Um programa feito em Python para detectar textos em imagens.
Todas as imagens processadas tem seus textos e detalhes guardados em um arquivo CSV dentro do projeto para o histórico de todos os processos.
No arquivo output_ocr.csv, estão gravados: nome, data de extração, texto extraído e caminho do arquivo.

*Nota: Para selecionar uma imagem no programa, primeiro garanta que ela está dentro da pasta /images, após isso, é possível selecionar seu index.*


## English
A program made with Python for detecting texts in images.
Every processed image text is written in a CSV file located in the project files for a history of all the processes.
In the output_ocr.csv file, the following data is written: name of the file, extraction date, extracted text and file path. 

*Note: To select an image in the program, first ensure that it is located in the /images folder, then, you can choose its index.*
